# Extensão de Navegador: Pesquisas Automáticas no Bing

## Função da Extensão
Esta extensão para navegador automatiza o processo de realização de pesquisas no Bing, simulando buscas tanto no modo desktop quanto no modo mobile. O objetivo é facilitar a obtenção de pontos em programas de recompensas, como o Microsoft Rewards, realizando buscas automaticamente com termos embaralhados e aleatórios.

## Como Usar
1. **Instalação**
   - Baixe ou clone este repositório.
   - No navegador Chrome, acesse `chrome://extensions/`.
   - Ative o modo "Developer" (Desenvolvedor).
   - Clique em "Load unpacked" (Carregar sem compactação) e selecione a pasta `extension` deste projeto.

2. **Configuração**
   - Abra o popup da extensão no navegador.
   - Defina a quantidade de buscas web (desktop) e mobile desejadas.
   - Ajuste o tempo de delay entre buscas, se necessário.
   - Clique para iniciar as pesquisas automáticas.

3. **Funcionamento**
   - A extensão abrirá novas abas para cada busca, alternando entre modo desktop e mobile conforme configurado.
   - O progresso pode ser acompanhado pelo popup.
   - Ao finalizar, todas as abas abertas pela extensão serão fechadas automaticamente.

## Observações
- É necessário estar logado em sua conta Microsoft para acumular pontos.
- Recomenda-se não usar o computador durante a execução para evitar interferências.
- O uso excessivo pode violar os termos de uso do Bing/Microsoft Rewards. Use por sua conta e risco.
